#####  Nombre del Proyecto: 
Calificación estudiantil

### Descripción

Este proyecto consiste en una aplicación de escritorio desarrollada en C# que permite registrar los datos de los estudiantes y sus calificaciones.
El sistema valida que los campos estén completos, calcula el promedio de las notas y muestra mensajes indicando si el estudiante aprobó o reprobó.

La aplicación fue realizada como práctica de aula, siguiendo las indicaciones explicadas en clase.

### Tecnologías Utilizadas

- Lenguaje: C#

- Framework: .NET (Windows Forms)

- Entorno de Desarrollo (IDE): Visual Studio

- Tipo de Aplicación: Escritorio
